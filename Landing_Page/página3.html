Explorando o ❄️ Mundo do Gelo ❄️ - Módulo Básico

Bem-vindo ao módulo "Básico" do nosso curso de logica de programação de ensino gamificado, onde você se aventurará em um mundo congelado. Neste módulo, revisaremos e aprofundaremos os conceitos essenciais de programação que você aprendeu no módulo Iniciante, mas agora em um ambiente gelado e desafiador.

Iglus do Conhecimento:

No módulo Básico, você aprimorará sua compreensão dos seguintes conceitos:

  • IDEs (Ambientes de Desenvolvimento Integrado): Programas que auxiliam a escrever, depurar e executar código de maneira eficiente.
  • Listas e Arrays: estruturas de dados usadas para armazenar coleções de elementos, como números ou palavras, em um único local.
  • Matrizes: São semelhantes a listas, mas organizam dados em duas dimensões, como uma tabela de linhas e colunas.
  • Ordenação de Dados: Um processo de organizar um conjunto de informações em uma ordem específica, como alfabética ou numérica.
  • Uso de Bibliotecas: Utilize um conjuntos de código pré-escrito, chamados bibliotecas, para adicionar funcionalidades específicas aos programas que você cria.

Projeto no Gelo ❄️:

Durante sua jornada congelante, você se depara com um desafio emocionante. Você está encarregado de escrever um programa que ajude a calcular a temperatura média em um mês de inverno no mundo do gelo. Isso ajudará os habitantes a se prepararem melhor para o clima extremo.

Aqui está um exemplo de código que pode ser usado para calcular a temperatura média:

OBS📝: Vamos supor que você escolheu a linguagem Python🐍 no mundo anterior

    
        # Declare uma lista para armazenar as temperaturas diárias
        temperatures = [0.0] * 31
        sum = 0.0
        
        print("Digite as temperaturas diárias do mês:")
        
        # Leia as temperaturas diárias e calcule a soma
        for i in range(31):
            temperature = float(input(f"Dia {i + 1}: "))
            temperatures[i] = temperature
            sum += temperature
        
        # Calcule a temperatura média
        average = sum / 31
        
        print(f"A temperatura média do mês é: {average}")

Seu desafio é implementar este algoritmo e verificar se ele funciona corretamente. Além disso, você pode aprimorar o programa adicionando recursos extras, como a exibição da temperatura mais alta e mais baixa do mês.
