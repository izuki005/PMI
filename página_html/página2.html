Explorando o 🏜️ Mundo do Deserto 🏜️ - Módulo Iniciante

Bem-vindo ao módulo "Iniciante" do nosso curso de lógica de programação de ensino gamificado, onde você começará sua jornada no mundo árido do Deserto 🌵. Assim como um viajante no deserto busca recursos para sobreviver, você explorará os conceitos fundamentais de programação neste ambiente desafiador usando a linguagem Portugol.

Oásis de Conhecimento:

Em sua jornada pelo Deserto, você encontrará diversos "oásis de conhecimento", representando conceitos vitais de programação. Alguns dos principais "oásis" que você explorará incluem:

  • Escopo de Programa: Aprenda a organizar e estruturar seu código.
  • Comandos de Seleção: Navegue pelas encruzilhadas das decisões em seu programa.
  • Variáveis: Descubra reservatórios para armazenar e manipular dados.
  • Tipos: Conheça os diferentes tipos de dados encontrados na programação.
  • Interação: Saiba como interagir com o usuário enquanto enfrenta temperaturas extremas.
  • Condição: Encontre refúgios de lógica para guiar seu caminho.
  • Repetição: Aprenda a superar obstáculos repetidamente.

Rastros no Deserto:

Para dar vida a esses conceitos, imagine que você é um aventureiro solitário no Deserto, seguindo rastros misteriosos. Aqui está um exemplo prático baseado nessa premissa: 

            
        Algoritmo SeguindoRastros
            Declare rastro1, rastro2, rastro3, descoberta: Real
        
            Escreva("Siga os rastros no Deserto:")
            Escreva("Rastro 1: ")
            Leia(rastro1)
        
            Escreva("Rastro 2: ")
            Leia(rastro2)
        
            Escreva("Rastro 3: ")
            Leia(rastro3)
        
            Se rastro1 > rastro2 E rastro1 > rastro3 Então
                descoberta <- "Rastro 1"
            Senão
                Se rastro2 > rastro1 E rastro2 > rastro3 Então
                    descoberta <- "Rastro 2"
                Senão
                    descoberta <- "Rastro 3"
                FimSe
            FimSe
        
            Escreva("Você descobriu o rastro de ", descoberta)
        FimAlgoritmo

Neste exemplo, você está seguindo rastros no Deserto e usando estruturas condicionais para determinar qual rastro seguir. Essa é uma analogia para o uso de estruturas de decisão em programação. Agora, logo abaixo você verá resoluções do mesmo problema em Python (🐍), C (🅒) e Java (☕).

Código em 🐍:

                       
                            rastro1 = float(input("Rastro 1: "))
                            rastro2 = float(input("Rastro 2: "))
                            rastro3 = float(input("Rastro 3: "))

                            if rastro1 > rastro2 and rastro1 > rastro3:
                                descoberta = "Rastro 1"
                            else:
                                if rastro2 > rastro1 and rastro2 > rastro3:
                                    descoberta = "Rastro 2"
                                else:
                                    descoberta = "Rastro 3"

                            print("Você descobriu o rastro de", descoberta)

                            #include

Código em ☕:

                        
                            import java.util.Scanner;

                            public class SeguindoRastros {
                                public static void main(String[] args) {
                                    Scanner input = new Scanner(System.in);
                                    float rastro1, rastro2, rastro3;
                                    String descoberta;

                                    System.out.print("Rastro 1: ");
                                    rastro1 = input.nextFloat();

                                    System.out.print("Rastro 2: ");
                                    rastro2 = input.nextFloat();

                                    System.out.print("Rastro 3: ");
                                    rastro3 = input.nextFloat();

                                    if (rastro1 > rastro2 && rastro1 > rastro3)
                                        descoberta = "Rastro 1";
                                    else {
                                        if (rastro2 > rastro1 && rastro2 > rastro3)
                                            descoberta = "Rastro 2";
                                        else
                                            descoberta = "Rastro 3";
                                    }

                                    System.out.println("Você descobriu o rastro de " + descoberta);
                                }
                            }

Escolha Sua Linguagem de Programação 📝:

Após concluir o módulo Iniciante no mundo do deserto, é hora de escolher a linguagem com a qual você deseja continuar sua jornada de programação. Temos por enquanto 3 linguagens disponíveis para ensino, cada uma com suas características únicas.

Escolha uma das seguintes linguagens e siga em frente:

  • Linguagem Python: Uma linguagem de programação versátil e popular, conhecida por sua sintaxe simples e legibilidade. É amplamente utilizada no desenvolvimento web, ciência de dados e automação.
  • Linguagem C: É uma linguagem poderosa que oferece controle de baixo nível, usada em sistemas embarcados, jogos e desenvolvimento de software de alto desempenho.
  • Linguagem Java: Uma linguagem amplamente utilizada em desenvolvimento de aplicativos Android e em empresas. Possui uma forte ênfase na orientação a objetos.

Após escolher sua linguagem, você continuará sua jornada de programação com base nela. Lembre-se de que cada linguagem oferece oportunidades e desafios exclusivos, então escolha aquela que mais se adequa aos seus objetivos e interesses.

Boa sorte com sua escolha e aproveite o próximo mundo da programação!
