Explorando a 🌲 Floresta 🌲 - Módulo Intermediário

Bem-vindo ao módulo Intermediário do nosso curso de lógica de programação de ensino gamificado, onde você entrará no mundo exuberante e denso da Floresta. Nesse ambiente envolvente, você irá construir sobre os conceitos fundamentais de programação que aprendeu nos módulos anteriores, usando a linguagem de programação escolhida no módulo Básico.

Desbravando a Floresta🌳:

No coração da Floresta, você encontrará uma série de desafios de programação e projetos que exigirão o conhecimento adquirido no módulo Básico. Alguns dos principais conceitos que você explorará aqui incluem:

  • Dominando Funções: Aprenda a encapsular código e criar blocos reutilizáveis.
  • Navegando com Ponteiros: Compreenda a gestão de memória por meio de ponteiros.

Missão na Floresta🌳:

Em sua jornada pela densa Floresta, você enfrentará um desafio intrigante. Sua tarefa é criar um programa que auxilie na catalogação de espécies de árvores na floresta. Isso ajudará os pesquisadores a estudar e conservar a biodiversidade da região.

OBS 📝: Vamos supor que você escolheu a linguagem Python no mundo do deserto

            
                # Declare uma lista para armazenar as espécies de árvores
                arvores = [None] * 100
                contador = 0
                
                print("Registre as espécies de árvores encontradas na Floresta:")
                
                # Leia as espécies de árvores e as armazene na lista
                while contador < 100:
                    especie = input(f"Espécie de árvore {contador + 1}: ")
                    arvores[contador] = especie
                    contador += 1
                    continuar = input("Deseja continuar? (S/N): ").upper()
                    if continuar != "S":
                        break
                
                print("Espécies de árvores catalogadas:")
                for i in range(contador):
                    print(f"{i + 1}: {arvores[i]}")

Seu desafio é implementar este algoritmo para catalogar as espécies de árvores na Floresta. Além disso, você pode melhorar o programa adicionando recursos adicionais, como a contagem de quantas vezes cada espécie de árvore foi encontrada.
